# ZBLS Generator

Un générateur et entraîneur d'algorithmes ZBLS (Zborowski-Bruchem Last Slot) pour la résolution du Rubik's Cube.

## Description

Ce projet contient :
- Une interface web d'entraînement (`ZB trainer.html`)
- Des bases de données d'algorithmes ZBLS et ZBLL
- Des scripts Python pour la génération et l'entraînement

## Fichiers principaux

- `ZB trainer.html` - Interface web principale pour l'entraînement
- `zbll_algorithms_database.py` - Base de données des algorithmes ZBLL
- `zbls_algorithms_database.py` - Base de données des algorithmes ZBLS
- `zbll_trainer.py` - Script d'entraînement Python
- `images/` - Images utilisées dans l'interface
- `svgs/` - Diagrammes SVG des cas ZBLS

## Utilisation

1. Ouvrir `ZB trainer.html` dans un navigateur web
2. Utiliser l'interface pour s'entraîner sur les différents cas ZBLS

## Dépendances

Python 3.x requis pour les scripts Python.
